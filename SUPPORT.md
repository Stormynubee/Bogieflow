# Support for Bogie Flow

Thank you for using Bogie Flow. This project was developed as a submission for the **FAR AWAY 2026 Hackathon** under the Railways theme.

## Support Channels

### 1. GitHub Issues
If you encounter a bug or have a feature suggestion, please check the existing issues before opening a new one.
- **Bug Reports**: Open a bug report if the application logic, telemetry loop, or UI renders incorrectly. Refer to our [Bug Report Template](.github/issue_template/bug_report.md).
- **Feature Requests**: Open a feature request if you have ideas on expanding the hydrology rules or hardware accelerometer integrations. Refer to our [Feature Request Template](.github/issue_template/feature_request.md).

### 2. Discussions
For general questions, configuration assistance, deployment questions, or feedback on our avoided-failure impact metrics, please use the **Discussions** tab on the GitHub repository.

### 3. Security Concerns
If you suspect a security issue (e.g. credential exposure, CORS policy bypass), please refer to our [Security Policy](SECURITY.md) and report it privately. **Do not create a public issue for security concerns.**
